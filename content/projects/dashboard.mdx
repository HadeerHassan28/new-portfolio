# Interactive Dashboard

A comprehensive dashboard application built with React and Vite, featuring interactive charts, data visualization, and a modern admin interface for displaying and analyzing data insights.

## Overview

This dashboard application provides a complete admin interface with various chart types, calendar functionality, and data management tools. Built with modern React practices and Material-UI, it offers a professional and responsive user experience.

## Key Features

- 📊 **Interactive Charts**: Multiple chart types using Nivo charts library
- 📅 **Calendar Integration**: Full calendar functionality with event management
- 🎛️ **Admin Interface**: Complete admin dashboard with sidebar navigation
- 📱 **Responsive Design**: Works perfectly on all device sizes
- 🎨 **Material Design**: Beautiful UI components with Material-UI
- 📋 **Form Validation**: Robust form handling with Formik and Yup
- 🧭 **Routing**: Smooth navigation with React Router
- 🎚️ **Pro Slider**: Advanced slider components

## Tech Stack

### Frontend Framework
- **Framework**: React.js with Vite
- **Language**: JavaScript/TypeScript
- **Routing**: React Router

### UI & Components
- **Design System**: Material-UI (MUI)
- **Charts**: Nivo Charts
- **Calendar**: FullCalendar
- **Slider**: React-Pro-Slider

### Form Management
- **Forms**: Formik
- **Validation**: Yup
