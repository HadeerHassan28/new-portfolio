# MessengerApp

A real-time chat application that allows users to sign in with their Google account, join chat rooms, and communicate with other users in the same room. Built with React and Firebase for seamless real-time messaging.

## Overview

MessengerApp provides a simple yet powerful chat experience where users can authenticate with Google, create or join chat rooms, and engage in real-time conversations. The application leverages Firebase for authentication, real-time database, and hosting.

## Key Features

- 🔐 **Google Authentication**: Secure sign-in with Google accounts
- 💬 **Real-time Messaging**: Instant message delivery and updates
- 🏠 **Room-based Chat**: Create and join different chat rooms
- 👥 **Multi-user Support**: Chat with multiple people simultaneously
- 📱 **Responsive Design**: Works perfectly on all devices
- 🔄 **Live Updates**: Messages appear instantly without refresh

## Tech Stack

### Frontend
- **Framework**: React.js with Vite
- **Styling**: Bootstrap
- **Language**: JavaScript

### Backend & Services
- **Authentication**: Firebase Auth (Google Provider)
- **Database**: Firebase Firestore
- **Hosting**: Firebase Hosting
- **Real-time**: Firebase Real-time Database
