# Contact List - React Native App

A mobile contact management application built with React Native as part of ITI coursework. The app provides a clean interface for displaying contact lists with detailed information for each contact, featuring modern Material-UI design and efficient data handling.

## Overview

This Contact List app demonstrates React Native development skills with a focus on clean UI design, efficient data management, and smooth navigation. Built using Material-UI components and modern React Native practices, it provides an intuitive contact browsing experience.
