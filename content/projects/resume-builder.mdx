# Resume Builder

A comprehensive web application that allows users to create professional resumes with an intuitive interface and download them as high-quality PDF files. Built with React frontend and Node.js backend.

## Overview

This resume builder provides a user-friendly interface for creating professional resumes. With real-time preview, multiple templates, and PDF generation capabilities, it simplifies the resume creation process for job seekers and professionals.

## Key Features

- 📄 **PDF Generation**: Download resumes as high-quality PDF files
- 👀 **Real-time Preview**: See changes instantly as you type
- 🎨 **Professional Templates**: Multiple resume layouts to choose from
- 📱 **Responsive Design**: Works seamlessly on all devices
- 💾 **Auto-save**: Never lose your progress
- 🖨️ **Print Ready**: Optimized for both digital and print formats

## Tech Stack

### Frontend
- **Framework**: React.js with Vite
- **Styling**: Bootstrap
- **Language**: JavaScript

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **PDF Generation**: Puppeteer
