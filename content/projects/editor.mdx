# Online Code Editor

A web-based code editor inspired by VSCode, built with React and Firebase. This editor provides essential coding features with cloud storage capabilities for a seamless development experience.

## Overview

This online code editor brings the familiar VSCode experience to the web browser. With Firebase integration for data persistence and a clean Bootstrap interface, it offers developers a convenient way to write, edit, and save code from anywhere.
