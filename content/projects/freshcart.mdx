# FreshCart E-commerce Platform

A comprehensive e-commerce web application that provides a complete online shopping experience with product browsing, cart management, secure payments, and user authentication. Built with modern React technologies and integrated with Stripe for payments.

## Overview

FreshCart is a full-stack e-commerce solution designed to provide users with a seamless online shopping experience. From product discovery to secure checkout, the platform handles all aspects of modern e-commerce with a focus on performance and user experience.

## Key Features

- 🛒 **Complete Shopping Cart**: Add, remove, and manage products
- 💳 **Secure Payments**: Stripe integration for safe transactions
- 🔐 **User Authentication**: JWT-based secure login system
- 📱 **Responsive Design**: Perfect experience on all devices
- 🔍 **Product Search**: Advanced search and filtering
- 📦 **Order Management**: Track orders and purchase history
- 🌐 **Offline Detection**: Handle offline scenarios gracefully
- 🎠 **Product Carousel**: Beautiful product showcases

## Tech Stack

### Frontend
- **Framework**: React.js with Vite
- **Styling**: Bootstrap
- **Language**: JavaScript

### Data & API
- **HTTP Client**: Axios
- **Custom Hooks**: useAxios for API management
- **Form Handling**: Formik
- **Validation**: Yup

### Features & Integrations
- **Authentication**: JWT-decode
- **Payments**: Stripe integration
- **Notifications**: Toast notifications
- **Carousel**: React Slick
- **Offline Detection**: react-detect-offline
