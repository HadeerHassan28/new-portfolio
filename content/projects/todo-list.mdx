# To-do List App

A simple and efficient to-do list application built with Angular, featuring a clean interface and essential task management functionality.

## Overview

This to-do list application provides users with an intuitive way to manage their daily tasks. Built with Angular and TypeScript, it offers a responsive design and smooth user experience with real-time notifications.

## Key Features

- ✅ **Add Tasks**: Easily create new tasks with a simple input interface
- ✏️ **Edit Tasks**: Modify existing tasks inline
- 🗑️ **Delete Tasks**: Remove completed or unwanted tasks
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices
- 🔔 **Toast Notifications**: Real-time feedback for user actions
- 💾 **Data Persistence**: Tasks are saved using JSON server

## Tech Stack

### Frontend
- **Framework**: Angular
- **Language**: TypeScript
- **Styling**: Bootstrap
- **Notifications**: ngx-toastr
- **ID Generation**: uuid

### Backend
- **Data Storage**: json-server (for development)
