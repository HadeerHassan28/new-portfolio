# Weather App

A clean and intuitive weather application that displays current weather conditions when users enter city and country names. Built with React and featuring form validation for a smooth user experience.

## Overview

This weather application provides users with current weather information for any city worldwide. With a simple form interface, robust validation, and responsive design, it offers a reliable way to check weather conditions anywhere.
