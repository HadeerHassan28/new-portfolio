# Lemon Squeezy

A sleek and modern landing page for a marketing startup, built with **Next.js** and **TypeScript**.  
The website highlights the company’s services with smooth animations, responsive layouts, and a clean user experience.

## Overview

The project was designed to deliver a professional and visually engaging portfolio site for a marketing company.  
It includes responsive layouts, service showcases, and interactive elements to ensure a smooth browsing experience across devices.

## Key Features

- 🎨 **Modern UI**: Clean and minimal design with smooth transitions
- 📱 **Responsive Layout**: Optimized for desktop, tablet, and mobile
- ⚡ **Next.js Performance**: Fast loading with static site generation
- 🌙 **Theming**: Support for light/dark mode
- 🧩 **Reusable Components**: Modular structure for easy updates and scalability

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + custom components
- **Deployment**: Vercel
