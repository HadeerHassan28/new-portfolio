# To Do List with Contact List App

A comprehensive React Native application that combines todo list management with contact list functionality. Built as an ITI project, this app demonstrates navigation between different app sections and efficient mobile development practices.

## Overview

This dual-purpose mobile application showcases the integration of two essential mobile app features: task management and contact organization. The app provides seamless navigation between todo management and contact browsing, demonstrating advanced React Native development skills.

## Key Features

- ✅ **Todo Management**: Create, edit, and manage tasks
- 👥 **Contact List**: Browse and view contact information
- 🧭 **Seamless Navigation**: Smooth transitions between app sections
- 📱 **Native Mobile Experience**: Optimized for mobile devices
- 🎨 **Clean Interface**: Intuitive and user-friendly design
- 🔄 **State Persistence**: Maintain data across app sessions
- 📊 **Efficient Data Handling**: Optimized performance

## Tech Stack

### Mobile Framework
- **Framework**: React Native
- **Language**: JavaScript
- **Navigation**: React Navigation

### Data & State
- **ID Generation**: UUID for unique identifiers
- **State Management**: React hooks and context
- **Data Persistence**: Local storage solutions
